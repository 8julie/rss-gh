## How to update

1. Update /config/urls
2. Run these:
```git tag
git tag build.url.<<<replace next number here>>>
git push --tags
```
1. Check (workflow)[https://github.com/8julie/rss-gh/actions] to see if changes were saved
2. Check (page)[https://8julie.github.io/rss-gh/#/] to see if page itself has been updated
